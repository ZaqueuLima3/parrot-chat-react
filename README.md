<h1 align="center">Parrot Chat</h1>

## Overview

Parrot chat is a challenge to construct a chat using websocket echo

## Roadmap

### Web version

- ReactJs
- Prop-types
- Styled-components
- Eslint

### Mobile version

- React Native
- Prop-types
- Styled-components
- Eslint

## Installation

```
$ git clone git@github.com:ZaqueuLima3/parrot-chat-react.git
```

### Web version

```
$ cd frontend && yarn
```

or

```
$ cd frontend && npm i
```

```
$ yarn start
```

or

```
$ npm start
```

### Mobile version

This pages will help you install and config your development environment

https://facebook.github.io/react-native/docs/getting-started.html
or
https://docs.rocketseat.dev/ambiente-react-native/introducao

```
$ cd mobile-one && yarn
```

or

```
$ cd mobile-one && npm i
```

```
$ react-native run-android
```

or

```
$ react-native run-ios
```
