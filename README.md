<h1 align="center">Parrot Chat</h1>

## Overview

Parrot chat is a challenge to construct a chat using websocket echo

## Roadmap

### Web version

- ReactJs
- Prop-types
- Styled-components
- Eslint

### Mobile version

- React Native
- Prop-types
- Styled-components
- Eslint

## Installation

```
$ git clone git@github.com:ZaqueuLima3/parrot-chat-react.git
```

### Web version

```
$ cd frontend && yarn
```

or

```
$ cd frontend && npm i
```

```
$ yarn start
```

or

```
$ npm start
```
